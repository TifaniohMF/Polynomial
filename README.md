# 🎯️ Polynomials

A polynomial is the form $P(X) = \sum_{i}^{n} a_{i}X^{i}$.
This repository permits to calculate polynomial rapidly.

## 📁️ Projects structure
```text
Polynomial/
├── docs
│   ├── polynomial.pdf          # support pdf
│   └── polynomial.tex
├── polynomial.py               # main code
├── README.md
└── test_polynomial.py          # Test 
```
## 💻️ Install and use
1. Clone repository
```text
git clone https://github.com/TifaniohMF/Polynomial.git
cd Polynomial
```
2. Execute
```python
python3 polynomial.py
```

## 🛠️ Prerequisites
Make you sure you have install:
- Python 3.x
- Git
